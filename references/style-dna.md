# 风格 DNA

## 一句话

纯白底、干净矢量线稿、单一品牌强调色、温暖肤色、真产品 UI 卡叠在线稿场景上，专业、可信、无花哨。

像一个 SaaS 品牌的官网首屏插画：**产品清晰可辨**，**氛围友好轻盈**，**技术感和亲和力平衡**。

## 必须

- 背景：**纯白** `#FFFFFF`。不允许米色、浅灰、渐变、纸纹、噪点、暖光。
- 主线稿：**纯黑** `#111111` 或深灰 `#1F1F1F`，矢量、干净、**均匀粗细**（约 2-2.5pt 在 1500px 宽度下），无手绘抖动、无粗细变化、无毛边、无双线。
- 强调色：**只有一个**品牌色（默认紫 `#7C5CFF`），实心填充，用于：
  - 装饰性聊天气泡的填充
  - UI 卡里的品牌图标 / CTA / 品牌按钮
  - 光标提示 pill 的边框或图标
  - 铅笔、火花等散落小装饰
  - **绝对不要**用强调色画阴影、渐变、光效
- 肤色：**单一温暖桃色** `#F4C7A8` 或 `#F1BFA6`，只填人物脸和手，不加腮红、不加高光、不加阴影。
- 头发：**纯黑实心色块**。眼睛/嘴：**极简线条或点**，2-3 笔完成。
- 版式：**主体占画面 55-70%**，画面周围留白，元素**居中略偏上**，底部有一条极细水平线暗示"桌面"或"地面"（可选）。

## 强调色调色板（备选品牌色）

如果用户给了品牌色 HEX 就用它。没给就从下面挑一个：

| 名字 | HEX | 情绪 |
|---|---|---|
| Signal Purple（默认） | `#7C5CFF` | 智能、AI、协作 |
| Focus Blue | `#3B6DFF` | 生产力、专业、金融 |
| Grow Green | `#22C55E` | 增长、健康、可持续 |
| Warm Coral | `#FF7A59` | 创意、社交、消费 |
| Deep Teal | `#0EA5A4` | 稳健、数据、B2B |
| Sunny Amber | `#F5A524` | 温暖、教育、生活 |

强调色只允许出现**一种**，不要混色。

## 绝对不要

- 不要渐变、发光、模糊、景深、光效
- 不要阴影（元素之间靠**层叠**表达前后关系，不靠 drop shadow）
- 不要纸纹、颗粒、复古滤镜、水彩、油画感
- 不要 3D 渲染、等距轴测（isometric）、C4D 感
- 不要手绘抖动线（这是本 skill 和 Ian 风格最大的区别）
- 不要多种强调色混用（不要"紫+绿+橙"这种彩虹装饰）
- 不要给人物加腮红、闪光眼、大眼睫毛、可爱表情包脸
- 不要真实照片素材、真实产品截图（要"仿真但矢量重画"的版本）
- 不要英文以外的水印文字（除非明确要求中文 UI）
- 不要写"AI-generated"字样、不要 emoji 装饰
- 不要复杂背景（书架、窗户、城市天际线）
- 不要在 UI 卡外圈加粗描边框（用元素本身的层次表达"卡"）

## 材质与光

- 全部**扁平**（flat），无光影、无高光、无反光
- 所有物体都是**面 + 线**的组合，绝不出现三维体积感
- 头像、笔电屏幕内的画面允许有极浅的品牌色淡填充作为"屏幕开着"的暗示（例如笔电屏幕背景是 `#F4F1FF` 一类的极浅紫），但不允许出现在 UI 卡外部

## 字体（在 UI 卡内部）

- 无衬线，接近 Inter / SF Pro / Söhne 的现代 UI 字
- 名字加粗，时间戳浅灰 `#9CA3AF`
- 正文黑色，段落间距舒适
- 不要用手写体、装饰体、衬线体
